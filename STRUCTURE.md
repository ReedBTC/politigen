## Quick Start

```bash
# Clone and enter the repo
git clone https://github.com/ReedBTC/politigen.git
cd politigen

# Install dependencies
pip install -r requirements.txt

# Refresh all data (optional — CSVs are already committed)
python collect_politigen.py

# View the site
open politigen.html        # macOS
xdg-open politigen.html    # Linux
# or just double-click the file
```

## Data Sources

| Source | What it provides |
|--------|-----------------|
| [unitedstates/congress-legislators](https://github.com/unitedstates/congress-legislators) | Every Congress member since 1789, with exact birth dates and term dates |
| [BLS CPS Table 18b](https://www.bls.gov/cps/cpsaat18b.xlsx) | Employed persons by industry and age bracket (annual average) |

The legislators source is fetched as two JSON files: `legislators-historical.json` (~15MB) and `legislators-current.json`. Both are downloaded in a single run of `collect_politigen.py` and are not stored in the repo.

## Generational Framework

All generation assignments use **Strauss-Howe** cutoffs (not Pew Research):

| Generation       | Birth Years |
|------------------|-------------|
| Pre-G.I.         | before 1901 |
| G.I. Generation  | 1901–1924   |
| Silent           | 1925–1942   |
| Baby Boom        | 1943–1960   |
| Gen X (13ers)    | 1961–1981   |
| Millennial       | 1982–2005   |
| Gen Z            | 2006+       |

Congressional data uses **exact birth dates** from the legislators JSON. BLS data uses age brackets, so generational shares are **approximated** by proportionally splitting BLS's 10-year brackets across generation boundaries.

## Data Flow

```
unitedstates/congress-legislators (GitHub)
    └── legislators-historical.json  ─┐
    └── legislators-current.json     ─┴─► collect_politigen.py ──► data/congress_historical.csv
                                                                 ──► data/congress_snapshots_detail.csv
                                                                 ──► data/bls_gen_comparison.csv
BLS.gov (CPS Table 18b Excel)
    └── cpsaat18b.xlsx ───────────────────► collect_politigen.py ──► data/bls_gen_comparison.csv

data/*.csv ───────────────────────────────► politigen.html  (currently hardcoded; CSV wiring TBD)
```

## What Was Removed

The original repo had `scripts/`, `charts/`, and `docs/` folders containing multiple
collection scripts, matplotlib chart generators, and several standalone HTML chart files.
All of that has been consolidated:

- 5 Python scripts → 1 (`collect_politigen.py`)
- Multiple chart HTML files → 1 (`politigen.html`)
- Intermediate/derived CSVs → removed (only final CSVs kept in `data/`)
- `docs/` (GitHub Pages) → removed (site will be served directly from `politigen.html`)
